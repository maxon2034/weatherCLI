# WeatherCLI ⛅

**WeatherCLI** — это интерактивное консольное приложение (CLI) на языке Go для получения актуальных прогнозов погоды в реальном времени. Программа использует бесплатное API **Open-Meteo** (без необходимости регистрации и получения API-ключей), поддерживает геокодирование городов, кэширование запросов для экономии трафика и встроенные механизмы повторных запросов при сетевых сбоях.

## 🌟 Описание возможностей

*   **Текущая сводка погоды**: Детальное отображение текущей температуры, субъективного ощущения (Feels Like), скорости и направления ветра, влажности воздуха, атмосферного давления, видимости и объема осадков.
*   **Интерактивное меню**: Удобная навигация без необходимости перезапуска приложения.
*   **Почасовой прогноз**: Наглядная таблица прогноза погоды на ближайшие 12 часов (шаг — 1 час) с отображением температуры, вероятности осадков и скорости ветра.
*   **Прогноз на неделю (7 дней)**: Компактная таблица на 7 дней с визуальными иконками погодных условий (солнце, облака, дождь, снег, гроза), минимальной/максимальной температурой и вероятностью осадков.
*   **Динамическая цветовая индикация (ANSI Colors)**: Температурные показатели автоматически окрашиваются в зависимости от погоды:
    *   `< 0°C` — ❄️ синий
    *   `0°C – 25°C` — ☀️ желтый
    *   `> 25°C` — 🔥 красный
*   **Умный In-Memory TTL-кэш**: Снижает нагрузку на сеть и API. Разные типы прогнозов имеют индивидуальное время жизни (TTL):
    *   Текущая погода: **5 минут**
    *   Почасовой прогноз: **15 минут**
    *   Прогноз на неделю: **30 минут**
*   **Смена города «на лету»**: Возможность быстро переключить город прямо из интерфейса с автоматической перезаписью в конфигурационный файл.
*   **Устойчивость к сбоям (Retry-механизм)**: Приложение автоматически делает до 3-х повторных попыток отправки HTTP-запросов с экспоненциальным увеличением задержки (Exponential Backoff), если API временно недоступно.

## 🛠 Используемые технологии

*   **Язык программирования**: Go (Golang) 1.21+[cite: 1, 4, 5, 9]
*   **Погодный провайдер**: [Open-Meteo API](https://open-meteo.com/) (Geocoding API для поиска координат по названию и Forecast API для получения метрик)[cite: 6]
*   **Формат конфигурации**: YAML (парсинг через библиотеку `gopkg.in/yaml.v3`)
*   **Конкурентность**: `sync.RWMutex` для безопасной многопоточной работы с кэшем из разных горутин[cite: 2]
*   **Контексты**: `context.Context` для управления жизненным циклом и таймаутами HTTP-запросов[cite: 1, 6, 9]

## 🚀 Установка и сборка

1. **Клонируйте репозиторий с проектом**:
    
    Bash
    
    ```
    git clone [https://github.com/yourusername/weather-cli.git](https://github.com/yourusername/weather-cli.git)
    cd weather-cli
    ```
    
2. **Загрузите необходимые зависимости** (пакет для работы с YAML):
    
    Bash
    
    ```
    go mod download
    ```
    
3. **Соберите исполняемый файл**:
    
    - **Для Linux / macOS**:
        
        Bash
        
        ```
        go build -o weather ./cmd/weathercli/main.go
        ```
        
    - **Для Windows**:
        
        Bash
        
        ```
        go build -o weather.exe ./cmd/weathercli/main.go
        ```
        
4. **Запуск**:
    
    - **Для Linux / macOS**: `./weather`
        
    - **Для Windows**: `weather.exe`
        

## ⚙️ Описание конфигурации (`config.yaml`)

Конфигурационный файл проекта имеет формат YAML и по умолчанию располагается (или создается) по пути, прописанному в `internal/config/config.go`.

> ⚠️ **Примечание**: Перед сборкой убедитесь, что путь к конфигурационному файлу в переменной `confFile` внутри `config.go` соответствует вашей системе, либо адаптируйте его под относительный путь[cite: 3].

### Структура файла `conf.yaml`:

YAML

```
default_city: "Minsk"
```

- `default_city` — город, для которого погода запрашивается автоматически при каждом старте приложения.
    
- Если файл отсутствует или поле пустое, программа при первом запуске вежливо попросит ввести город в консоли и сохранит его для последующих сессий.
    

## 📖 Руководство по использованию

После запуска приложения перед вами появится текущая сводка для вашего города по умолчанию, а затем интерактивное меню ожидания команд:

Plaintext

```
> [Введите команду и нажмите Enter]
```

### Доступные интерактивные команды:

- `1` — Выводит **почасовой прогноз** на ближайшие 12 часов в табличном виде.
    
- `2` — Выводит **прогноз на неделю** (7 дней) с погодными условиями и иконками.
    
- `c` (или `C`) — **Сменить город**. Вас попросят ввести новое название (например, `Madrid` или `Tokyo`). Приложение тут же покажет погоду для нового города и перезапишет его в файл конфигурации как город по умолчанию.
    
- `r` (или `R`) — **Принудительно обновить данные** (если данные были в кэше, они обновятся из сети).
    
- `q` (или `Q`) — Безопасный **выход** из программы.
